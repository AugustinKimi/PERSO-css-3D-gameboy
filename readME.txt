Bonjour, 

Voila le jeu snake sur gameboy classic. 

Vous avez le choix de la couleur dans le menu pour personnalisé votre gameboy.

Pour jouer au jeu snake qui est touta  fait classic il suffit d'utiliser les fleches directionnelles, les touches z q s d ou  bien cliquer sur les boutons de la crois sur la gameboy directement. le jeu s'accelère au fur et a mesure que vous mangez des pommes.
